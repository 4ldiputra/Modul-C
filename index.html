🚀 Pengenalan Algoritma & Bahasa C

Apa itu Algoritma?

Algoritma adalah serangkaian langkah-langkah logis yang disusun secara sistematis untuk memecahkan suatu masalah atau mencapai tujuan tertentu.

Karakteristik Algoritma:
  • Input: Memiliki nol atau lebih masukan
  • Output: Memiliki satu atau lebih keluaran
  • Definiteness: Setiap langkah harus jelas dan tidak ambigu
  • Finiteness: Harus berakhir dalam jumlah langkah terbatas
  • Effectiveness: Setiap langkah harus dapat dikerjakan

Pengenalan Bahasa C

Bahasa C adalah bahasa pemrograman tingkat menengah yang dikembangkan oleh Dennis Ritchie pada tahun 1972. C menjadi dasar bagi banyak bahasa pemrograman modern.

Program C Pertama:

#include 

int main() {
    printf("Hello, World!\n");
    return 0;
}
Penjelasan Kode:
• #include <stdio.h> - Menyertakan header file untuk input/output
• int main() - Fungsi utama program
• printf() - Fungsi untuk mencetak ke layar
• return 0; - Mengakhiri program dengan status sukses

🔧 Function & Recursion

Melanjutkan Pass by Reference

Contoh Lengkap Pass by Value vs Reference:
#include #include // Pass by value - nilai asli tidak berubah void ubahNilai(int x) { x = 100; printf("Dalam function ubahNilai: x = %d\n", x); } // Pass by reference - menggunakan pointer void ubahNilaiReference(int *x) { *x = 100; printf("Dalam function ubahNilaiReference: *x = %d\n", *x); } // Function untuk swap dua variabel void swap(int *a, int *b) { int temp = *a; *a = *b; *b = temp; } // Function untuk mengisi array void isiArray(int arr[], int ukuran) { for (int i = 0; i < ukuran; i++) { arr[i] = i * 10; } } int main() { int angka = 50; printf("Sebelum ubahNilai: angka = %d\n", angka); ubahNilai(angka); printf("Setelah ubahNilai: angka = %d\n\n", angka); printf("Sebelum ubahNilaiReference: angka = %d\n", angka); ubahNilaiReference(&angka); printf("Setelah ubahNilaiReference: angka = %d\n\n", angka); // Contoh swap int x = 10, y = 20; printf("Sebelum swap: x = %d, y = %d\n", x, y); swap(&x, &y); printf("Setelah swap: x = %d, y = %d\n\n", x, y); // Contoh dengan array int data[5]; isiArray(data, 5); printf("Array setelah diisi:\n"); for (int i = 0; i < 5; i++) { printf("data[%d] = %d\n", i, data[i]); } return 0; }

Recursion (Rekursi)

Rekursi adalah teknik di mana function memanggil dirinya sendiri untuk menyelesaikan masalah yang lebih kecil.

Contoh Rekursi - Faktorial:
#include // Rekursi untuk faktorial int faktorial(int n) { // Base case if (n <= 1) { return 1; } // Recursive case return n * faktorial(n - 1); } // Rekursi untuk Fibonacci int fibonacci(int n) { if (n <= 1) { return n; } return fibonacci(n - 1) + fibonacci(n - 2); } // Rekursi untuk menghitung pangkat int pangkat(int base, int exp) { if (exp == 0) { return 1; } return base * pangkat(base, exp - 1); } int main() { int n = 5; printf("Faktorial %d = %d\n", n, faktorial(n)); printf("Deret Fibonacci:\n"); for (int i = 0; i < 10; i++) { printf("%d ", fibonacci(i)); } printf("\n\n"); printf("2^5 = %d\n", pangkat(2, 5)); return 0; }

🏗️ Structure & Union

Struktur (struct)

Struktur memungkinkan kita mengelompokkan berbagai tipe data menjadi satu unit.

Contoh Struktur Mahasiswa:
#include #include // Definisi struktur struct Mahasiswa { int nim; char nama[50]; float ipk; char jurusan[30]; }; // Typedef untuk mempermudah penggunaan typedef struct { int hari; int bulan; int tahun; } Tanggal; typedef struct { int nim; char nama[50]; Tanggal tanggalLahir; float ipk; } DataMahasiswa; int main() { // Deklarasi dan inisialisasi struktur struct Mahasiswa mhs1; mhs1.nim = 12345; strcpy(mhs1.nama, "Budi Santoso"); mhs1.ipk = 3.75; strcpy(mhs1.jurusan, "Informatika"); // Inisialisasi langsung struct Mahasiswa mhs2 = {54321, "Sari Dewi", 3.85, "Sistem Informasi"}; // Menggunakan typedef DataMahasiswa mhs3; mhs3.nim = 67890; strcpy(mhs3.nama, "Andi Wijaya"); mhs3.tanggalLahir.hari = 15; mhs3.tanggalLahir.bulan = 8; mhs3.tanggalLahir.tahun = 2000; mhs3.ipk = 3.92; // Menampilkan data printf("Data Mahasiswa 1:\n"); printf("NIM: %d\n", mhs1.nim); printf("Nama: %s\n", mhs1.nama); printf("IPK: %.2f\n", mhs1.ipk); printf("Jurusan: %s\n\n", mhs1.jurusan); printf("Data Mahasiswa 3:\n"); printf("NIM: %d\n", mhs3.nim); printf("Nama: %s\n", mhs3.nama); printf("Tanggal Lahir: %d/%d/%d\n", mhs3.tanggalLahir.hari, mhs3.tanggalLahir.bulan, mhs3.tanggalLahir.tahun); printf("IPK: %.2f\n", mhs3.ipk); return 0; }

Union

Union mirip dengan struct, tetapi semua anggotanya berbagi lokasi memori yang sama.

Contoh Union:
#include // Union untuk menyimpan berbagai tipe data union Data { int i; float f; char c; }; typedef union { struct { unsigned char byte1; unsigned char byte2; unsigned char byte3; unsigned char byte4; } bytes; unsigned int value; } IntBytes; int main() { union Data data; printf("Ukuran union Data: %lu bytes\n", sizeof(data)); // Menggunakan union data.i = 42; printf("data.i = %d\n", data.i); data.f = 3.14; printf("data.f = %.2f\n", data.f); printf("data.i sekarang = %d (corrupt)\n", data.i); data.c = 'A'; printf("data.c = %c\n", data.c); printf("data.f sekarang = %.2f (corrupt)\n", data.f); // Contoh praktis - memecah integer menjadi bytes printf("\nMemecah integer menjadi bytes:\n"); IntBytes intData; intData.value = 0x12345678; printf("Nilai: 0x%08X\n", intData.value); printf("Byte 1: 0x%02X\n", intData.bytes.byte1); printf("Byte 2: 0x%02X\n", intData.bytes.byte2); printf("Byte 3: 0x%02X\n", intData.bytes.byte3); printf("Byte 4: 0x%02X\n", intData.bytes.byte4); return 0; }

💾 Memory Allocation

Dynamic Memory Allocation

Dynamic memory allocation memungkinkan program mengalokasikan memori saat runtime.

malloc, calloc, realloc, dan free:
#include #include #include int main() { int n; printf("Masukkan jumlah elemen: "); scanf("%d", &n); // malloc - mengalokasikan memori tanpa inisialisasi int *arr1 = (int*)malloc(n * sizeof(int)); if (arr1 == NULL) { printf("Alokasi memori gagal!\n"); return 1; } // calloc - mengalokasikan memori dengan inisialisasi 0 int *arr2 = (int*)calloc(n, sizeof(int)); if (arr2 == NULL) { printf("Alokasi memori gagal!\n"); free(arr1); return 1; } // Mengisi array printf("Masukkan %d angka:\n", n); for (int i = 0; i < n; i++) { scanf("%d", &arr1[i]); } // Menampilkan array printf("Array 1 (malloc): "); for (int i = 0; i < n; i++) { printf("%d ", arr1[i]); } printf("\n"); printf("Array 2 (calloc): "); for (int i = 0; i < n; i++) { printf("%d ", arr2[i]); } printf("\n"); // realloc - mengubah ukuran memori printf("Mengubah ukuran array menjadi %d elemen\n", n * 2); arr1 = (int*)realloc(arr1, n * 2 * sizeof(int)); if (arr1 == NULL) { printf("Realloc gagal!\n"); free(arr2); return 1; } // Inisialisasi elemen baru for (int i = n; i < n * 2; i++) { arr1[i] = i * 10; } printf("Array setelah realloc: "); for (int i = 0; i < n * 2; i++) { printf("%d ", arr1[i]); } printf("\n"); // Jangan lupa free memory free(arr1); free(arr2); return 0; }

Memory Management untuk Struktur

Dynamic Array of Structures:
#include #include #include typedef struct { int id; char nama[50]; float nilai; } Siswa; Siswa* buatArraySiswa(int jumlah) { Siswa *siswa = (Siswa*)malloc(jumlah * sizeof(Siswa)); if (siswa == NULL) { printf("Alokasi memori gagal!\n"); return NULL; } return siswa; } void inputSiswa(Siswa *siswa, int index) { printf("Masukkan data siswa ke-%d:\n", index + 1); printf("ID: "); scanf("%d", &siswa[index].id); printf("Nama: "); scanf("%s", siswa[index].nama); printf("Nilai: "); scanf("%f", &siswa[index].nilai); } void tampilkanSiswa(Siswa *siswa, int jumlah) { printf("\nData Siswa:\n"); printf("%-5s %-20s %-10s\n", "ID", "Nama", "Nilai"); printf("------------------------------------\n"); for (int i = 0; i < jumlah; i++) { printf("%-5d %-20s %-10.2f\n", siswa[i].id, siswa[i].nama, siswa[i].nilai); } } int main() { int jumlah; printf("Masukkan jumlah siswa: "); scanf("%d", &jumlah); Siswa *siswa = buatArraySiswa(jumlah); if (siswa == NULL) { return 1; } for (int i = 0; i < jumlah; i++) { inputSiswa(siswa, i); } tampilkanSiswa(siswa, jumlah); free(siswa); return 0; }

#include #include #include typedef struct { int nim; char nama[50]; float ipk; char jurusan[30]; } Mahasiswa; void simpanMahasiswa(Mahasiswa mhs[], int jumlah, const char *filename) { FILE *file = fopen(filename, "wb"); if (file == NULL) { printf("Tidak dapat membuat file %s!\n", filename); return; } fwrite(mhs, sizeof(Mahasiswa), jumlah, file); fclose(file); printf("Data berhasil disimpan ke %s\n", filename); } int bacaMahasiswa(Mahasiswa mhs[], int maxJumlah, const char *filename) { FILE *file = fopen(filename, "rb"); if (file == NULL) { printf("File %s tidak ditemukan!\n", filename); return 0; } int jumlah = fread(mhs, sizeof(Mahasiswa), maxJumlah, file); fclose(file); return jumlah; } void tampilkanMahasiswa(Mahasiswa mhs[], int jumlah) { printf("\nData Mahasiswa:\n"); printf("%-10s %-20s %-8s %-15s\n", "NIM", "Nama", "IPK", "Jurusan"); printf("--------------------------------------------------------\n"); for (int i = 0; i < jumlah; i++) { printf("%-10d %-20s %-8.2f %-15s\n", mhs[i].nim, mhs[i].nama, mhs[i].ipk, mhs[i].jurusan); } } int main() { Mahasiswa mahasiswa[100]; int jumlah = 0; int pilihan; do { printf("\n=== SISTEM DATA MAHASISWA ===\n"); printf("1. Input data mahasiswa\n"); printf("2. Tampilkan data\n"); printf("3. Simpan ke file\n"); printf("4. Baca dari file\n"); printf("5. Keluar\n"); printf("Pilihan: "); scanf("%d", &pilihan); switch(pilihan) { case 1: if (jumlah < 100) { printf("Masukkan data mahasiswa ke-%d:\n", jumlah + 1); printf("NIM: "); scanf("%d", &mahasiswa[jumlah].nim); printf("Nama: "); scanf("%s", mahasiswa[jumlah].nama); printf("IPK: "); scanf("%f", &mahasiswa[jumlah].ipk); printf("Jurusan: "); scanf("%s", mahasiswa[jumlah].jurusan); jumlah++; } else { printf("Data sudah penuh!\n"); } break; case 2: if (jumlah > 0) { tampilkanMahasiswa(mahasiswa, jumlah); } else { printf("Belum ada data!\n"); } break; case 3: if (jumlah > 0) { simpanMahasiswa(mahasiswa, jumlah, "mahasiswa.dat"); } else { printf("Belum ada data untuk disimpan!\n"); } break; case 4: jumlah = bacaMahasiswa(mahasiswa, 100, "mahasiswa.dat"); if (jumlah > 0) { printf("Berhasil membaca %d data mahasiswa\n", jumlah); } break; } } while (pilihan != 5); return 0; }

🔢 Algoritma Sorting

Bubble Sort

Bubble sort adalah algoritma sorting sederhana yang membandingkan elemen bersebelahan dan menukarnya jika tidak berurutan.

Implementasi Bubble Sort:
#include void bubbleSort(int arr[], int n) { int temp; for (int i = 0; i < n - 1; i++) { for (int j = 0; j < n - i - 1; j++) { if (arr[j] > arr[j + 1]) { // Swap elements temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } } } } void printArray(int arr[], int size) { for (int i = 0; i < size; i++) { printf("%d ", arr[i]); } printf("\n"); } int main() { int arr[] = {64, 34, 25, 12, 22, 11, 90}; int n = sizeof(arr)/sizeof(arr[0]); printf("Array sebelum sorting: "); printArray(arr, n); bubbleSort(arr, n); printf("Array setelah bubble sort: "); printArray(arr, n); return 0; }

Selection Sort

Implementasi Selection Sort:
#include void selectionSort(int arr[], int n) { int minIndex, temp; for (int i = 0; i < n - 1; i++) { minIndex = i; // Cari elemen terkecil di sisa array for (int j = i + 1; j < n; j++) { if (arr[j] < arr[minIndex]) { minIndex = j; } } // Tukar elemen if (minIndex != i) { temp = arr[i]; arr[i] = arr[minIndex]; arr[minIndex] = temp; } } } int main() { int arr[] = {29, 10, 14, 37, 13, 25, 6}; int n = sizeof(arr)/sizeof(arr[0]); printf("Array sebelum sorting: "); for (int i = 0; i < n; i++) { printf("%d ", arr[i]); } printf("\n"); selectionSort(arr, n); printf("Array setelah selection sort: "); for (int i = 0; i < n; i++) { printf("%d ", arr[i]); } printf("\n"); return 0; }

Insertion Sort

Implementasi Insertion Sort:
#include void insertionSort(int arr[], int n) { int key, j; for (int i = 1; i < n; i++) { key = arr[i]; j = i - 1; // Pindahkan elemen yang lebih besar dari key while (j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j = j - 1; } arr[j + 1] = key; } } int main() { int arr[] = {5, 2, 4, 6, 1, 3}; int n = sizeof(arr)/sizeof(arr[0]); printf("Array sebelum sorting: "); for (int i = 0; i < n; i++) { printf("%d ", arr[i]); } printf("\n"); insertionSort(arr, n); printf("Array setelah insertion sort: "); for (int i = 0; i < n; i++) { printf("%d ", arr[i]); } printf("\n"); return 0; }

Error Handling

Error Handling yang Baik:
#include #include #include #include // Function untuk safe division int safeDivide(int dividend, int divisor, float *result) { if (result == NULL) { fprintf(stderr, "Error: NULL pointer untuk result\n"); return -1; } if (divisor == 0) { fprintf(stderr, "Error: Division by zero\n"); return -1; } *result = (float)dividend / divisor; return 0; } // Function untuk safe file operation FILE* safeFileOpen(const char *filename, const char *mode) { FILE *file = fopen(filename, mode); if (file == NULL) { fprintf(stderr, "Error: Tidak dapat membuka file '%s' dengan mode '%s'\n", filename, mode); fprintf(stderr, "System error: %s\n", strerror(errno)); } return file; } int main() { float result; // Test safe division if (safeDivide(10, 2, &result) == 0) { printf("10 / 2 = %.2f\n", result); } if (safeDivide(10, 0, &result) != 0) { printf("Division by zero handled gracefully\n"); } // Test safe file operation FILE *file = safeFileOpen("test.txt", "w"); if (file != NULL) { fprintf(file, "Hello, World!\n"); fclose(file); printf("File berhasil ditulis\n"); } // Test reading non-existent file file = safeFileOpen("nonexistent.txt", "r"); if (file == NULL) { printf("File tidak ditemukan - error handled\n"); } return 0; }

Best Practices

Tips Pemrograman C yang Baik:
• Selalu inisialisasi variabel sebelum digunakan
• Gunakan const untuk nilai yang tidak berubah
• Periksa return value dari function
• Selalu free() memory yang di-malloc()
• Gunakan meaningful variable names
• Tambahkan komentar untuk kode yang kompleks
• Hindari global variables jika memungkinkan
• Gunakan static untuk function yang hanya dipakai dalam file yang sama
Common Mistakes yang Harus Dihindari:
• Buffer overflow - selalu cek batas array
• Memory leaks - jangan lupa free()
• Uninitialized variables
• Infinite loops
• Integer overflow
• Null pointer dereference

Perulangan for digunakan ketika kita tahu berapa kali perulangan akan dijalankan.

Contoh for loop:
#include int main() { // Menampilkan angka 1-10 printf("Angka 1 sampai 10:\n"); for (int i = 1; i <= 10; i++) { printf("%d ", i); } printf("\n\n"); // Menampilkan angka genap printf("Angka genap 2 sampai 20:\n"); for (int i = 2; i <= 20; i += 2) { printf("%d ", i); } printf("\n\n"); // Countdown printf("Countdown:\n"); for (int i = 10; i >= 1; i--) { printf("%d ", i); } printf("Start!\n"); return 0; }

📊 Array & Pointer

Array Satu Dimensi

Array adalah kumpulan elemen dengan tipe data yang sama, disimpan dalam lokasi memori yang berurutan.

Deklarasi dan Inisialisasi Array:
#include int main() { // Berbagai cara deklarasi array int angka[5]; // Array kosong dengan 5 elemen int nilai[] = {80, 90, 75, 88, 92}; // Ukuran otomatis int data[5] = {1, 2, 3}; // Sisanya akan 0 // Mengisi array printf("Masukkan 5 angka:\n"); for (int i = 0; i < 5; i++) { printf("Angka ke-%d: ", i+1); scanf("%d", &angka[i]); } // Menampilkan array printf("\nAngka yang dimasukkan:\n"); for (int i = 0; i < 5; i++) { printf("angka[%d] = %d\n", i, angka[i]); } // Mencari nilai maksimum int max = angka[0]; for (int i = 1; i < 5; i++) { if (angka[i] > max) { max = angka[i]; } } printf("\nNilai maksimum: %d\n", max); return 0; }

Pengenalan Pointer

Pointer adalah variabel yang menyimpan alamat memori dari variabel lain.

Dasar-dasar Pointer:
#include int main() { int angka = 42; int *ptr; // Deklarasi pointer ptr = &angka; // ptr menunjuk ke alamat angka printf("Nilai angka: %d\n", angka); printf("Alamat angka: %p\n", &angka); printf("Nilai ptr: %p\n", ptr); printf("Nilai yang ditunjuk ptr: %d\n", *ptr); // Mengubah nilai melalui pointer *ptr = 100; printf("\nSetelah *ptr = 100:\n"); printf("Nilai angka: %d\n", angka); printf("Nilai yang ditunjuk ptr: %d\n", *ptr); // Pointer dan array int arr[5] = {10, 20, 30, 40, 50}; int *arr_ptr = arr; // Nama array adalah pointer ke elemen pertama printf("\nArray menggunakan pointer:\n"); for (int i = 0; i < 5; i++) { printf("arr[%d] = %d, *(arr_ptr + %d) = %d\n", i, arr[i], i, *(arr_ptr + i)); } return 0; }
