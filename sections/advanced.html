📌 Proyek Akhir: Program Pengelolaan Inventaris Barang

Buatlah sebuah program dalam bahasa C untuk mengelola data inventaris barang di sebuah toko. Setiap barang memiliki informasi sebagai berikut:

  • Kode barang (integer)
  • Nama barang (string)
  • Jumlah barang (integer)
  • Satuan (misalnya: pcs, box, liter)
  • Harga per unit (double)

Program harus menyimpan data barang menggunakan struct dan array serta menyediakan menu:

  • Tambah barang baru
  • Hapus barang berdasarkan kode
  • Ubah data barang
  • Cari barang berdasarkan kode
  • Tampilkan seluruh data inventaris
  • Keluar dari program

Saat user memilih menu Tambah Barang, program harus meminta input semua data barang dan menyimpannya jika masih ada ruang. Jika memilih Hapus Barang, program harus menghapus barang berdasarkan kode dan merapikan isi array. Program berjalan terus hingga user memilih menu Keluar.
