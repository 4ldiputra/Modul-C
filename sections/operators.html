⚡ Operator & Aritmatika

Operator Aritmatika

Operator Operasi Contoh Hasil
+Penjumlahan5 + 38
-Pengurangan5 - 32
*Perkalian5 * 315
/Pembagian10 / 33 (integer)
%Modulus (sisa bagi)10 % 31
Program Kalkulator Sederhana:

#include <stdio.h>

int main() {
    int a = 15, b = 4;
    
    printf("a = %d, b = %d\n", a, b);
    printf("a + b = %d\n", a + b);
    printf("a - b = %d\n", a - b);
    printf("a * b = %d\n", a * b);
    printf("a / b = %d\n", a / b);        // Pembagian integer
    printf("a / b = %.2f\n", (float)a / b); // Pembagian float
    printf("a %% b = %d\n", a % b);       // Modulus
    
    return 0;
}

Operator Perbandingan

Operator Operasi Contoh Hasil
==Sama dengan5 == 5True (1)
!=Tidak sama dengan5 != 3True (1)
>Lebih besar5 > 3True (1)
<Lebih kecil3 < 5True (1)
>=Lebih besar atau sama dengan5 >= 5True (1)
<=Lebih kecil atau sama dengan3 <= 5True (1)
Contoh Program Perbandingan:

#include <stdio.h>

int main() {
    int x = 7, y = 10;

    printf("x == y : %d\n", x == y);
    printf("x != y : %d\n", x != y);
    printf("x > y  : %d\n", x > y);
    printf("x < y  : %d\n", x < y);
    printf("x >= y : %d\n", x >= y);
    printf("x <= y : %d\n", x <= y);

    return 0;
}

Operator Logika

Operator Nama Hasil
&& AND true jika kedua kondisi true
|| OR true jika salah satu kondisi true
! NOT Membalikkan nilai kebenaran
Contoh Program Operator Logika

#include <iostream>
using namespace std;

int main() {
    bool x = true, y = false;

    cout << "=== Operator Logika ===" << endl;
    cout << "x && y : " << (x && y) << endl; // AND
    cout << "x || y : " << (x || y) << endl; // OR
    cout << "!x     : " << (!x) << endl;     // NOT
    cout << "!y     : " << (!y) << endl;

    return 0;
}

🎯 Latihan Soal Operator

  1. Hitung hasil dari operasi berikut dan tampilkan di layar:
    • 25 + 10
    • 25 - 10
    • 25 * 10
    • 25 / 10
    • 25 % 10
  2. Diberikan variabel a = 7 dan b = 3. Cetak hasil perbandingan berikut:
    • a == b
    • a != b
    • a > b
    • a <= b
  3. Diberikan x = true dan y = false. Tampilkan hasil dari:
    • x && y
    • x || y
    • !x
    • !y
  4. Buat program kalkulator sederhana yang:
    • Membuat dua variabel integer, misalnya a = 15 dan b = 4
    • Tampilkan hasil penjumlahan, pengurangan, perkalian, pembagian
