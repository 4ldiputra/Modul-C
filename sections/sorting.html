Konsep: Bubble Sort membandingkan dua elemen berdekatan dan menukarnya jika urutannya salah. Proses ini diulang hingga tidak ada lagi pertukaran yang dilakukan.

Analogi: Seperti gelembung air yang naik ke permukaan - elemen terbesar akan "mengapung" ke posisi terakhir.

ğŸ“‹ Cara Kerja Bubble Sort

Langkah-langkah:

  1. Bandingkan elemen pertama dengan elemen kedua
  2. Jika elemen pertama lebih besar, tukar posisinya
  3. Lanjutkan ke pasangan berikutnya
  4. Ulangi proses untuk semua elemen
  5. Setelah satu putaran, elemen terbesar akan berada di posisi terakhir
  6. Ulangi untuk elemen-elemen sebelumnya

ğŸ’» Implementasi Bubble Sort dalam C

#include <stdio.h>

void bubbleSort(int arr[], int n) {
    int i, j, temp;
    int tukar; // flag untuk optimasi
    
    // Loop untuk setiap putaran
    for (i = 0; i < n - 1; i++) {
        tukar = 0; // reset flag
        
        // Loop untuk membandingkan elemen berdekatan
        for (j = 0; j < n - i - 1; j++) {
            // Jika elemen kiri lebih besar, tukar
            if (arr[j] > arr[j + 1]) {
                temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                tukar = 1;
            }
        }
        
        // Jika tidak ada pertukaran, array sudah terurut
        if (tukar == 0) {
            break;
        }
    }
}

void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr[] = {5, 2, 8, 1, 9};
    int n = sizeof(arr) / sizeof(arr[0]);
    
    printf("Array sebelum sorting: ");
    printArray(arr, n);
    
    bubbleSort(arr, n);
    
    printf("Array setelah sorting: ");
    printArray(arr, n);
    
    return 0;
}

Kompleksitas Bubble Sort:

  • Best Case: O(n) - ketika data sudah terurut
  • Average Case: O(nÂ²)
  • Worst Case: O(nÂ²) - ketika data terurut terbalik
  • Space Complexity: O(1) - tidak butuh memori tambahan

ğŸ¯ 2. Selection Sort

Konsep: Selection Sort mencari elemen terkecil dan menempatkannya di posisi pertama, lalu mencari elemen terkecil berikutnya untuk posisi kedua, dan seterusnya.

Analogi: Seperti memilih pemain terpendek untuk baris paling depan, lalu yang lebih tinggi di belakangnya.

ğŸ“‹ Cara Kerja Selection Sort

Langkah-langkah:

  1. Cari elemen terkecil dalam seluruh array
  2. Tukar dengan elemen di posisi pertama
  3. Cari elemen terkecil dalam sisa array (kecuali posisi 1)
  4. Tukar dengan elemen di posisi kedua
  5. Ulangi hingga semua elemen terurut

#include <stdio.h>

void selectionSort(int arr[], int n) {
    int i, j, minIndex, temp;
    
    // Loop untuk setiap posisi
    for (i = 0; i < n - 1; i++) {
        // Anggap elemen pertama adalah minimum
        minIndex = i;
        
        // Cari elemen terkecil di sisa array
        for (j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        
        // Tukar elemen minimum dengan elemen di posisi i
        if (minIndex != i) {
            temp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = temp;
        }
    }
}

void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr[] = {5, 2, 8, 1, 9};
    int n = sizeof(arr) / sizeof(arr[0]);
    
    printf("Array sebelum sorting: ");
    printArray(arr, n);
    
    selectionSort(arr, n);
    
    printf("Array setelah sorting: ");
    printArray(arr, n);
    
    return 0;
}

Kompleksitas Selection Sort:

  • Best Case: O(nÂ²) - selalu melakukan perbandingan penuh
  • Average Case: O(nÂ²)
  • Worst Case: O(nÂ²)
  • Space Complexity: O(1)
  • Keuntungan: Jumlah pertukaran minimal (maksimal n-1 kali)

ğŸƒ 3. Insertion Sort

Konsep: Insertion Sort membangun array terurut satu elemen pada satu waktu dengan menyisipkan setiap elemen baru ke posisi yang tepat.

Analogi: Seperti menyusun kartu di tangan - ambil satu kartu, lalu sisipkan ke posisi yang benar di antara kartu yang sudah tersusun.

ğŸ“‹ Cara Kerja Insertion Sort

Langkah-langkah:

  1. Mulai dari elemen kedua (anggap elemen pertama sudah terurut)
  2. Simpan elemen saat ini sebagai "key"
  3. Bandingkan key dengan elemen-elemen sebelumnya
  4. Geser elemen yang lebih besar dari key ke kanan
  5. Sisipkan key di posisi yang tepat
  6. Ulangi untuk semua elemen

ğŸ’» Implementasi Insertion Sort dalam C

#include <stdio.h>

void insertionSort(int arr[], int n) {
    int i, j, key;
    
    // Mulai dari elemen kedua
    for (i = 1; i < n; i++) {
        key = arr[i];  // Simpan elemen yang akan disisipkan
        j = i - 1;
        
        // Geser elemen yang lebih besar dari key ke kanan
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j = j - 1;
        }
        
        // Sisipkan key di posisi yang tepat
        arr[j + 1] = key;
    }
}

void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr[] = {5, 2, 8, 1, 9};
    int n = sizeof(arr) / sizeof(arr[0]);
    
    printf("Array sebelum sorting: ");
    printArray(arr, n);
    
    insertionSort(arr, n);
    
    printf("Array setelah sorting: ");
    printArray(arr, n);
    
    return 0;
}

Kompleksitas Insertion Sort:

  • Best Case: O(n) - ketika data sudah terurut (hanya perbandingan)
  • Average Case: O(nÂ²)
  • Worst Case: O(nÂ²) - ketika data terurut terbalik
  • Space Complexity: O(1)
  • Keuntungan: Sangat efisien untuk data yang hampir terurut

ğŸ† Kapan Menggunakan Algoritma Apa?

  • Bubble Sort: Untuk pembelajaran dan data sangat kecil (<10 elemen)
  • Selection Sort: Ketika memori swap mahal (embedded systems)
  • Insertion Sort: Untuk data yang hampir terurut atau input real-time
  • Untuk data besar: Gunakan Quick Sort, Merge Sort, atau Heap Sort
